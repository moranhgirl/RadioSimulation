>> 本程序将 sitePlanV3 表格以 C++ 进行实现，用于对隧道内信号波的反射位置进行计算，通过输入隧道和电台参数得到反射位置，作为电台架设的位置参考。本程序通过对手动分段后的每段隧道单独进行计算，再进行加权平均计算得到整段隧道的平均落点以及平均电台信息。

>> 隧道的分段参考隧道侧视图，以一个坡为一段作为分段基准，平地作为坡的特殊情况——0 度坡处理，也算作一段。

>> 程序运行后优先请求隧道分段的段数，然后对每段单独请求隧道和电台参数。
请求参数包括以下内容：
隧道的长度、电台频率、电台带宽、弯道半径、弯道长度及此段隧道和海平面的角度。如果没有弯道，默认弯道半径和长度为 0，不做请求。

>> 输出结果包括以下内容：
隧道所有反射点的平均距离、每段隧道最近反射点的加权平均距离、每段隧道最远反射点的加权平均距离、所有反射后的带宽加权平均值、信号延迟的加权平均值、所有反射后的信号强度加权平均值。

>> 本程序包括以下文件：main.cpp、one_hop.cpp、one_hop.h、stdev.cpp、stdev.h
main.cpp：用于从键盘请求参数，调用 one_hop.cpp 计算每段隧道的数据，对所有数据加权平均计算并输出结果。
one_hop.cpp：用于对每段隧道进行数据计算，可得到每段隧道上下左右分别的最近、最远落点、落点平均距离、反射后的信号强度、反射后带宽、反射后的延迟、反射后的信号分贝。
one_hop.h：one_hop.cpp 的头文件，声明函数
stdev.cpp：计算样板标准方差的函数，由 one_hop.cpp 进行调用
stdev.h：stdev.cpp 的头文件，声明函数

>> 程序会检测参数是否输入错误并请求重新输入，每次只能修改一个数值，每次修改都会再次请求确认参数是否正确。但如果输入数据类型错误程序可能会直接退出，需要重开程序，请注意不要输入错数据类型。
